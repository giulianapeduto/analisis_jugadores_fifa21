# Análisis de jugadores FIFA 21

Proyecto de análisis exploratorio de datos utilizando el dataset de jugadores de FIFA 21.

## Objetivo

Explorar y analizar información sobre jugadores, clubes y valores de mercado para identificar patrones y tendencias dentro del dataset.

## Herramientas utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Análisis realizados

- Top 10 jugadores más valiosos
- Países con mayor cantidad de jugadores
- Clubes con mayor valor total
- Valor promedio por posición
- Relación entre edad y valor de mercado

## Dataset

Dataset público de jugadores FIFA 21.
